# Deploy na Vercel

Este projeto já está organizado para deploy direto na Vercel.

## Passos
1. Envie o conteúdo deste ZIP para a raiz do repositório.
2. Na Vercel, importe o repositório do GitHub.
3. O framework será detectado como Next.js automaticamente.
4. Se houver variáveis de ambiente, cadastre em Settings > Environment Variables.
5. Clique em Deploy.

## Importante
- A pasta `app/` está na raiz do projeto.
- O arquivo `vercel.json` já foi incluído.
- O arquivo `index.html` não é necessário para a Vercel em projetos Next.js.
